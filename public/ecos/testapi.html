Cómo usar la API

Para obtener los datos de Firestore, realiza una solicitud GET a:

http://localhost/ecos2/api.php

Para especificar una ruta personalizada:

http://localhost/ecos2/api.php?path=tu/ruta/personalizada

Por defecto, la API devuelve los datos de:

artifacts/naturaleza-sintetica/public/data/interactions
